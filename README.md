# Michael Strever Portfolio

A minimal static portfolio site for Michael Strever, focused on immersive music
and sound work. The visual system uses dark concrete tones, maroon accents, and
a scroll-responsive brutalist corridor background.

## Files

- `index.html` - page structure and portfolio copy
- `styles.css` - brutalist visual system, responsive layout, and 3D corridor
- `script.js` - scroll-based depth and panel motion

Open `index.html` directly in a browser, or serve the folder with any static
file server.
